# HR Analytics Dashboard (Power BI)

##  Business Objective
The purpose of this project is to build a complete HR Analytics dashboard that consolidates KPIs related to attrition, performance, compensation, satisfaction and absenteeism.

This dashboard enables HR Leaders to quickly monitor workforce health and make dataâ€‘driven decisions.

## Dataset
Source: HR Employee Attrition Dataset (Kaggle)  
Includes 34 HR variables used to build:
- HR KPIs
- Attrition metrics
- Compensation analysis
- Satisfaction insights
- Workforce risk indicators

##  Tools Used
- Power BI Desktop
- DAX (Measures, Calculated Columns)
- Data Modeling (Star Schema)
- Interactive Visuals

## Dashboard Pages
1. **Overview**
   - Total employees
   - Attrition
   - Salary analysis
   - Satisfaction scores

2. **Attrition Detail**
   - Attrition by department, job role, age, salary, gender
   - Attrition heatmap

3. **Performance & Satisfaction**
   - Job satisfaction levels
   - Environment satisfaction
   - Work-life balance indicators

4. **Compensation & Overtime**
   - Salary distribution
   - Overtime impact

5. **Absenteeism & Risk**
   - High-risk groups
   - Predictive indicators
