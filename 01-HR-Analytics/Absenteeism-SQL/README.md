# Absenteeism Analysis (SQL)

## Business Objective
This project aims to analyze absenteeism patterns to help HR teams identify productivity risks, attendance issues, and teams or roles with high absenteeism levels.

##  Dataset
Source: HR Employee Attrition Dataset (Kaggle)  
Using fields related to:
- Work-life balance
- Environment satisfaction
- Job involvement
- Overtime
- Distance from home
- Years in current role
- Attrition status

##  Tools Used
- SQL (SQLite / MySQL / SQL Server)
- Aggregations, GROUP BY, JOIN, CASE WHEN

##  Key SQL Queries
- Absence indicators by department
- Absence patterns by job role
- Overtime vs absenteeism
- Distance from home vs absenteeism
- Satisfaction scores vs absenteeism
- High-risk employee segments

##  Insights
- Departments with the highest absenteeism
- How satisfaction affects attendance
- Impact of commute distance on absences
- Whether overtime predicts unexplained absences
